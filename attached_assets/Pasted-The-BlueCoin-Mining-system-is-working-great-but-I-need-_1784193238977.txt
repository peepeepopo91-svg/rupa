## 1. Fix Daily Exchange Limit Counter

The daily exchange counter is displaying decimal values.

Example:

1.2222222222222223 / 3

This is incorrect.

The counter should always display whole numbers only.

Example:

Before any exchanges:
3 / 3

After 1 exchange:
2 / 3

After 2 exchanges:
1 / 3

After 3 exchanges:
0 / 3

Do not use floating point calculations for the remaining exchange count.

Track exchanges as integers.

The exchange limit should still reset every new day.

## 2. Fix Block Mining Countdown

The block mining countdown is currently updating only once every 10 seconds.

This makes the timer jump from values like:

59:50
59:40
59:30

Instead of:

59:59
59:58
59:57
59:56
...

Fix the timer so it updates every single second.

Requirements:

- Update once per second.
- Smooth countdown.
- Every user should see the same timer.
- The timer must continue using the shared server-side/fixed block timer.
- No jumping or skipping seconds.

The Admin Panel must have its own secure login.

Username

Password 1

Password 2

The admin can only log in if all three values are correct.

Store these credentials in a separate YAML configuration file so they are easy to change later.

Example:

  username: admin
  password1: firstPassword
  password2: secondPassword

Do NOT reuse the normal user login.

This is a completely separate administrator authentication system.
