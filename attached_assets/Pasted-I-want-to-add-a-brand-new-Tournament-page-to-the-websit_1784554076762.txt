I want to add a brand-new **Tournament** page to the website. This should be a **Grand Page**, not a basic page. It should become the complete hub for all PvP tournaments hosted on my Minecraft server.

The design should feel like a professional esports tournament platform, with modern animations, premium UI, interactive brackets, live updates, and full integration with the Admin Panel.

The system should support both **players** and **administrators**, allowing players to view and participate in tournaments while giving administrators complete control over tournament management.

---

# 1. Tournament Home

Create a modern landing page for the current tournament.

Display:

* Tournament banner
* Tournament name
* Tournament description
* Tournament status

  * Upcoming
  * Registration Open
  * Registration Closed
  * Live
  * Completed
* Countdown timer
* Registration deadline
* Tournament start date
* Prize pool
* Supported gamemode(s)
* Server IP
* Total registered teams
* Total registered players
* Register button
* Tournament rules button

If no tournament is active, display a clean message indicating that there are currently no active tournaments.

Support:

* Double Elimination (future-ready)

* Grand Finals

# 17. Performance

Optimize the system for large tournaments.

Support:

* Hundreds of players
* Many teams
* Many completed tournaments

Use pagination for any new lists or tables, following the project standard of **10 items per page** where applicable.

---

# 18. Security

Everything must be server-authoritative.

Players must never be able to:

* Modify brackets
* Change scores
* Register after registration closes
* Edit tournament data
* View admin-only controls
* Manipulate tournament results

Validate every action on the server.

---

# 19. GitHub Backup

All tournament-related data should use the existing automatic GitHub backup system.

Do not create a separate backup system.

---

# Goal

Create a professional, feature-rich **Tournament Grand Page** that feels like a real esports platform. Players should be able to register, view live brackets, follow tournament progress, check schedules, read announcements, view past tournaments, and track player/team statistics. Administrators should have a powerful **Tournament Management Grand Section** with complete control over every aspect of tournament creation, brackets, matches, registrations, prizes, and announcements, all synchronized live and integrated with the existing website infrastructure.
